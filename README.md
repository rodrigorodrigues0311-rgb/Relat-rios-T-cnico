# CSN Mineração – Sistema de Relatórios de Automação

App PWA para geração de relatórios técnicos da Equipe de Automação da CSN Mineração.

## 📱 Como instalar no Android

1. Acesse o link do GitHub Pages no **Chrome** do Android
2. Aguarde alguns segundos — aparecerá o banner **"Adicionar CSN Automação à tela inicial"**
3. Toque em **Instalar** → o app aparece na tela inicial como um app nativo!

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `csn-automacao`)
2. Faça upload de todos os arquivos desta pasta
3. Vá em **Settings → Pages → Source → main branch → / (root)**
4. Aguarde ~1 min e acesse: `https://SEU_USUARIO.github.io/csn-automacao/`

## 📋 Funcionalidades

- Identificação do relatório (empresa, equipe, data, DDS)
- Tipo de atividade: Em Oportunidade ou Preventiva
- Processos: Embarque de Minério, Descarga de Minério, Recarga de Carvão
- Atividades realizadas com anexo de fotos
- Lista de colaboradores
- Atividades não realizadas / desvios
- Pendências encontradas
- **Geração de PDF** com compartilhamento nativo (WhatsApp, e-mail, etc.)
